# Influencer 行动路线图（Yuxun）

## 🧭 阶段 1：明确定位（Week 1）
- 定义领域与受众
- 编写一句话定位（Slogan）
- 确定三大支柱主题（AI 实战工具 / 工程思维 / 学习成长）

## 🧩 阶段 2：内容引擎（Week 2）
- 建立 /content 目录结构
- 每篇内容使用 /templates/content-sop.md 模板
- 每周输出 3 微内容 + 1 长文 + 1 模板产物

## ⚙️ 阶段 3：飞轮运转（Week 3–4）
- 在 README 添加最新内容索引
- 每周复盘 docs/weekly/<date>.md
- 设定衡量指标：每周发布≥4 篇内容，新增 1 模板；每月 Star+10、粉丝+50

## 🌱 阶段 4：放大与合作（Month 2–3）
- 建立 docs/collab.md 追踪合作
- README 增加作品集区块
- 每月底复盘：本月我学到的 5 件事

## 🚀 阶段 5：影响力变现（Month 3+）
- 软变现：Upwork/Fiverr、Patreon、课程
- 硬变现：咨询、工具推广
- 建立 Newsletter 或社群（如 Substack / 微信群）

## 📅 执行节奏
| 周数 | 重点任务 | 结果 |
|------|-----------|------|
| Week 1 | 定位 & 方向确定 | README + overview 完成 |
| Week 2 | 内容生产机制建立 | 模板 + 3 条内容 |
| Week 3 | 启动内容飞轮 | weekly report + 索引页 |
| Week 4 | 复盘 & 调整方向 | 数据分析 + 调整计划 |
| Month 2 | 放大影响力 | 合作 & 项目 |
| Month 3 | 开始变现 | 课程 / 咨询 / Patreon |
