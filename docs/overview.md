# Influencer Roadmap (详细版)  

## 定位  
- 选择细分领域，明确目标受众及其需求。  
- 提炼价值主张（USP），回答“为什么追随你？”。  
- 研究竞争对手，总结差异化策略。  

## 内容支柱  
- 确定 3–4 个内容支柱，每个支柱围绕不同维度（教育、娱乐、实用、观点等）。  
- 为每个支柱制定主题列表和创作角度。  
- 确保支柱间协同，一致构建个人品牌形象。  

## 30 天冲刺计划  
- 制定每日或每周的内容日历，包括主题、形式和渠道。  
- 关注试验与复盘：每周分析表现数据（曝光、互动、转化）并调整策略。  
- 持续积累素材库和灵感来源。  

## 内容生产 SOP  
- **灵感收集**：建立素材库和灵感列表，使用工具记录（Notion、备忘录等）。  
- **选题与策划**：根据支柱和热点选题，编写提纲或脚本。  
- **创作与编辑**：按既定格式创作内容，并进行剪辑或排版。  
- **发布与推广**：根据渠道调整发布时间与文案，使用标签和话题提高可见度。  
- **互动与复盘**：及时回复评论，整理反馈，为下一周期优化。  

## 常见工具  
- 规划与写作：Notion、Trello、Google Docs  
- 设计与排版：Canva、Figma  
- 数据分析：平台自带分析、Google Analytics  
- 自动化与发布：Buffer、Later、Zapier
