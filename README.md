# Influencer Playbook  

欢迎来到 Influencer Playbook 仓库。本仓库包含一份路线图和模板，帮助你成为一名有影响力的人。  

## 路线图  

1. **定位 (Positioning)**  
   - 明确你的定位：选择细分领域并定义你的目标受众和价值主张。  
   - 分析竞品和目标人群，找出差异化优势。  

2. **内容支柱 (Content Pillars)**  
   - 选择 3-4 个与你品牌相关的主题作为内容支柱。  
   - 每个支柱应围绕目标受众的需求和兴趣制定。  

3. **30 天冲刺 (30-Day Sprint)**  
   - 制定 30 天的内容发布计划，保持频率和一致性。  
   - 利用不同平台测试形式（图文、短视频、长文、直播等），根据数据进行迭代。  
   - 每周复盘，分析效果并优化策略。  

4. **内容生产 SOP (Content SOP)**  
   - 明确流程：选题 → 规划 → 创作 → 编辑 → 发布 → 复盘。  
   - 使用模板提升效率，比如脚本模板、标题生成规则、互动话术等。  
   - 保持品牌统一性，如视觉风格、语调和标签。  

## 仓库结构  

- `README.md` – 项目简介和路线图。  
- `docs/overview.md` – 详细路线图和行动指南。  
- `docs/content_sop.md` – 内容生产 SOP 模板。  
- `.github/ISSUE_TEMPLATE.md` – Issue 模板，方便提交建议或问题。  
- `.github/PULL_REQUEST_TEMPLATE.md` – PR 模板，规范贡献流程。  
- `CONTRIBUTING.md` – 贡献指南。  

## 如何使用  

1. 阅读本仓库中的路线图和模板，根据自身情况进行调整。  
2. 使用提供的 Issue 和 PR 模板管理内容创作和团队协作。  
3. 分享你的成果并持续迭代，成为你领域的影响者！
